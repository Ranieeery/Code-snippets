# Node First Steps

## 1. Install Node.js

- [Node.js](https://nodejs.org/en/)

## 2. Node init

```bash
npm init
```

Sempre que quiser criar um projeto com as configurações padrões, use o comando abaixo:

```bash
npm init -y
```

## 3. Node run

```bash
node index.js
```

## 4. Install Typescript

```bash
npm install -g typescript
tsc --init
```
